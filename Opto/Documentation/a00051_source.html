7 // Header for the O22SnapIoMemMap C++ class.
8 //
9 // The O22SnapIoMemMap C++ class is used to communicate from a computer to an
10 // Opto 22 OptoMMP device.
11 //
12 // The basic procedure for using this class is:
13 //
14 // 1. Create an instance of the O22SnapIoMemMap class
15 // 2. Call OpenEnet() or OpenEnet2() to start connecting to an I / O unit
16 // 3. Call IsOpenDone() to complete the connection process
17 // 4. Call SetCommOptions() to set the desired timeout value
18 // 5. Configure the I / O(scaling, point types, ranges, baud rates, etc) using
19 // like ConfigurePt().
20 // 6. Call any of the memory map functions, such as GetAnaPtValue(),
21 // SetDigBankPointStates(), and ReadBlock(), to communicate with the I / O unit.
22 // Repeat as neccessary.
23 // 7. Call Close() when completely fnished messaging the device or a fault
24 // occurred.Applications typically operate in a loop(step 6),
25 // regularly accessing the device.
26 //
27 // While this code was developed on Microsoft Windows targeting x86 operating
28 // systems, it is intended to be as generic as possible. For Windows specific
29 // code, search for "_WIN32". For Linux specific code, search for "_LINUX".
30 // We validated this implementation on Ubuntu 16.04 desktop 64-bit with
31 // the gnu compiler collection.
